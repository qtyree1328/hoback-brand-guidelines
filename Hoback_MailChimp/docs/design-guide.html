Email Button Code Pattern

Use this table-based "bulletproof button" pattern in email templates. This renders correctly in every email client including Outlook:

<table role="presentation" cellspacing="0" cellpadding="0" border="0"> <tr> <td style="background-color: #ede9e1; padding: 12px 32px;"> <a href="#" style="font-family: 'Lato', Helvetica, Arial, sans-serif; font-size: 11px; font-weight: 700; color: #1e2d3a; text-decoration: none; text-transform: uppercase; letter-spacing: 3px; display: inline-block;"> Reserve Your Seat </a> </td> </tr> </table>

Swap the background-color and color values to create any variant.

Implementation Notes

  • Use table-based layout only — flexbox, grid, and div positioning break in Outlook
  • Use the bulletproof button pattern (table > tr > td with background-color) for all buttons
  • Google Fonts render in Apple Mail, iOS Mail, and Gmail app. Falls back to Georgia/Helvetica in Outlook
  • SVG may not render in Outlook desktop — the shield logo uses PNG to avoid this
  • All images need explicit width attributes and alt text
  • Upload event photos at 1200 x 600px for retina displays
  • Keep total email HTML under 102KB to avoid Gmail clipping
  • The drawings use inline opacity: 0.15 (mountains) and opacity: 0.25 (building) to stay subtle
  • Inline all CSS — Mailchimp's auto-inliner handles <style> blocks but some clients strip them
  • To use the template: Content → Email Templates → Create Template → Paste in code
